The canonical source

"You never leave the agent for a static tool. You talk to it. It does the work." The agent's toolkit isn't a static configuration you set up once; it evolves, under your direction, in response to what the work actually needs.

What the thesis means

The agent (Claude, in a Claude Code session) is the primary tool. Every other piece of deskwork — CLI verbs, the studio, doctor rules, sidecars, the calendar — exists to be read, edited, and orchestrated by that agent.

B · scrapbook sheet open · 65% height · 2-column grid of cards · "open viewer ↗" link in head

PatternThe bar grows from three tabs to four in review mode. Edit mode keeps the established three (Format / Notes / Save). The scrapbook tab uses a grid glyph (▦) — visual shorthand for "a contact sheet of supporting materials" — and a kraft-tone count badge that distinguishes it from the red-pencil Notes count.

Sheet content2-column card grid; each card is a thumbnail + filename + format chip. Image cards render an abstract press-color preview (red ↘ blue gradient on cream). Text cards (`.md`) render with horizontal-rule paper texture + a pilcrow glyph. Audio cards (`.mp3`) render with a waveform-evoking radial pattern. The thumbnail style is the differentiation: each format gets its own editorial visual treatment instead of generic file-icon-and-extension chrome.

Sheet head"5 items · open viewer ↗" — the count is at-a-glance, the standalone-viewer link is one tap away. The viewer link is proof-blue (consistent with Source/Preview accents and the existing scrapbook drawer's "open viewer" affordance on desktop).

StrongPredictable. Same idiom as the existing tabs. Discovery is permanent — the operator sees the count badge at-a-glance even when not in the sheet.

Weak97.5px per tab is tighter than the 130px of the 3-tab layout. The labels still fit (longest is "Scrapbook" at 9 chars, mono 0.6rem, ~70px) but the visual feels denser. The 4th tab dilutes the "thumb-reach for decisions" framing of the original Mockup 1 — Decisions / Outline / Notes were 3 peers; now they share the bar with a fourth thing that's about the entry rather than an action on it.
