Published view · scrapbook

iterate is missing from Final by design (Commandment II — Final locks content; iterate would mutate). It's also missing from Blocked / Cancelled / Published — those stages have no active in-place revision. block pauses an in-pipeline entry without abandoning it (waiting on external review, etc.); induct… teleports to an operator-chosen stage (works in both directions — forward to skip stages, or backward to retreat to a prior stage). Both are secondary verbs by frequency, so they live in the overflow menu but not in the swipe drawer. The … on induct indicates a sub-prompt: the operator picks which stage to teleport to.

Desktop affordance (1100px+). Same vocabulary, different chrome — high-frequency verbs stay inline as outlined chips; ⋮ menu adds block / induct… / cancel / scrap (secondary verbs). Operator gains the new verbs without crowding the row.
