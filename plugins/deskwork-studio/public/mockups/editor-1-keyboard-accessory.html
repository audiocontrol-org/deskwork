# Deskwork architectural thesis
 
Read this before touching any deskwork code.

## The canonical source
 
> "You never leave the agent for a static tool. You talk to it. It does the work."
 
The agent's toolkit isn't a static configuration you set up once; it evolves, under your direction, in response to what the work actually needs.
 
## What the thesis means
 
The agent (Claude, in a Claude Code session) is the primary tool. Every other piece of deskwork — CLI verbs, the studio, doctor rules, sidecars, the calendar — exists to be read, edited, and orchestrated by that agent.

B · typing · accessory bar above keyboard · save pill turns green

PatternTwo pieces of UI exist at rest — a circular back glyph and a Save state pill. The pill is faded gray ("Saved · 9:38") when clean and stamp-green with a glowing dot ("Save") when dirty. Tapping it writes to disk. The whole document is 92% of the visible canvas.

When typingThe iOS keyboard rises. A 48px keyboard accessory bar rides directly above it — H1 / H2 / B / I / code / link / list / quote — markdown shortcuts at thumb reach. Tapping a shortcut wraps the selection or inserts at the caret without leaving the editor. The undo button sits at the right edge.

Mode toggleSource / Preview as a two-state pill on the bottom strip (when keyboard is down). No Split — split-pane is a desktop luxury that crushes on phone (#239). Preview swaps the entire pane in place; the operator scrubs Source ↔ Preview as a binary, never a tri-state.

Notes & DiscardThe bottom strip's right side has two text-affordances: Notes (3) opens the same Notes sheet pattern from the review surface (continuity), and Discard exits edit mode. No "Cancel" button competing with Save — Discard is the explicit verb for "throw away this work."

Why this is rightThe current edit toolbar (#239) crams 9 chrome elements into a single mobile row that wraps to two lines. This direction goes the other way: chrome shrinks to two affordances. Markdown helpers (the actual editing power) move to where the thumb is — above the keyboard. The press-check editorial language is under-stated, not absent: paper texture, mono labels, italic display in the back glyph, red caret, stamp-green save state. The page becomes a writing surface.
