CSF-1 Token alignment harmful

content.css defines parallel --paper / --ink / --accent tokens with subtly different hex values than the editorial-print tokens. Operators see palette drift moving between surfaces. Fix: delete the parallel definitions and let --er-* flow through.

▲ Current — drift
paper / editorial#F5F1E8
paper / writers-cat#faf6ee
red-pencil#B8362A
accent (oxblood)#6e2e2e

Same token name, different hex. The cream is yellower in the content view; the red is darker.

● Proposed — single source
paper#F5F1E8
paper-2#ECE6D4
red-pencil#B8362A
red-soft#D9867A

One palette across all surfaces. Marginalia is one red. Paper is one cream.

CSF-2 Pixel spacing → tokens harmful

35 spacing-in-px declarations in content.css. Replace with --er-space-*.

}
body {
  padding: 0 var(--er-space-4) var(--er-space-6);
}
