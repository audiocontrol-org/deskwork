# Rule-bracketed stamps

## What

A fourth rubber-stamp variant: state labels rendered between thin horizontal rules ("[ IN REVIEW ]" / "[ ITERATING ]") at the row's leading edge. The bracketed-rule treatment was meant to feel typographic rather than graphic — a printer's mark instead of a stamp.

## Why rejected

Same root cause as the other rubber-stamp variants ([rotated](../2026-05-09-rotated-rubber-stamps-on-mobile/), [filing-tabs](../2026-05-09-filing-tab-stamps-on-mobile/), [letterpress-tags](../2026-05-09-letterpress-tag-stamps/)): the labels surfaced review state, which is RETIRED. Different graphical treatment, same underlying conceit, same retirement.

Rule-bracketed marks have legitimate uses elsewhere in the press-check vocabulary (printer's marks on the entry-review surface, masthead ornaments). The rejection is scoped to using them as state-chrome on dashboard rows.

## When
