# Letterpress-tag stamps

## What

Another variant of the rubber-stamp conceit: a letterpress-tag block — boxy, debossed-looking, monospace label — anchored to the row's leading edge. Same state labels (IN REVIEW / ITERATING / APPROVED) — but rendered as a letterpress impression instead of a rubber stamp or a filing tab.

## Why rejected

Same root cause as [the rotated rubber-stamps](../2026-05-09-rotated-rubber-stamps-on-mobile/) and [the filing-tabs](../2026-05-09-filing-tab-stamps-on-mobile/): the labels surfaced review state, which is RETIRED. The rubber-stamp conceit on mobile is retired in ALL its graphical variants — rotated stamps, letterpress tags, filing tabs, rule-bracketed marks, embossed seals. Switching the graphic style without changing the underlying "state-on-a-rectangle as the row's visual primary" pattern was not a meaningful change.

This was the second iteration after [the rotated stamps](../2026-05-09-rotated-rubber-stamps-on-mobile/) had been rejected, and it triggered the operator's "if we have to relitigate basic design decisions, we'll never make any headway" framing — which in turn led to writing `DESIGN-STANDARDS.md` and this archive.

## When

Rejected 2026-05-09. Mockup was originally drafted as `dashboard-1b-press-tabs-revised.html`; moved into this archive entry as the canonical visual.
