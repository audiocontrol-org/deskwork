# Stage filter chips

## What

A horizontal row of filter chips at the top of the mobile dashboard (one chip per stage), tap-to-include / tap-again-to-exclude. The intent was to let the operator quickly narrow the dashboard to a subset of stages — e.g., "show me only Outlining and Drafting."

## Why rejected

The operator never used filter chips on prior dashboards even when they were available — every dashboard task ended up either (a) viewing the whole pipeline shape or (b) drilling into one stage's expanded tile. The "filter to a subset" use case is real but doesn't justify a persistent UI affordance: the collapsible-tiles direction makes single-stage focus a single tap (expand the tile you want, leave others collapsed), without adding a filter row.

## When

Rejected 2026-05-09 during the dashboard exploration. Did not reach a mockup; conceptual rejection only.
