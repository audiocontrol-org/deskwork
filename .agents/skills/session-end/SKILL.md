# Session End

1. Update the feature `README.md`.
2. Update `workplan.md`.
3. Append a `DEVELOPMENT-NOTES.md` entry using the journal format from `AGENTS.md`.
4. If the session exercised deskwork or dw-lifecycle in earnest, append a `USAGE-JOURNAL.md` entry.
5. Update or close GitHub issues as appropriate.
6. Commit the documentation changes.
