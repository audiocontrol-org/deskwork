# Feature Extend

1. Read the existing PRD, workplan, and README.
2. Interview for the new capability and why it is being added now.
3. Propose new phases and acceptance criteria.
4. Update the PRD and workplan by appending, not rewriting.
5. Ensure the PRD has a `deskwork.id`.
6. Re-run the deskwork review/iterate/approve flow for the PRD.
7. Do not file new issues until the PRD workflow is `applied`.
8. After approval, file/update GitHub issues and report the additions.
