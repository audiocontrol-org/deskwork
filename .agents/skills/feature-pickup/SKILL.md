# Feature Pickup

1. Identify the feature from the worktree/branch.
2. Read:
   - `docs/1.0/001-IN-PROGRESS/<slug>/workplan.md`
   - `docs/1.0/001-IN-PROGRESS/<slug>/README.md`
   - the latest `DEVELOPMENT-NOTES.md` entry
3. Check open GitHub issues for the feature.
4. Report:
   - current phase
   - progress
   - next task
   - open issues
   - proposed approach
5. Wait for user confirmation before coding.
